# Moodish (무디시) — 서비스 기획서

> 기분 기반 AI 요리 추천 서비스 | 감성 푸드 다이어리

- **서비스명**: Moodish (무디시)
- **의미**: mood + -ish ("기분에 따른", "기분스러운")
- **도메인 후보**: moodish.kr / getmoodish.com
- **슬로건 후보**: "오늘 기분, 어떤 맛?" / "기분대로 한 끼"

---

## 1. 서비스 개요

### 컨셉

사용자의 오늘 기분과 상황을 입력하면, AI가 그에 어울리는 한국형 레시피를 감성 메시지와 함께 생성해주는 서비스. 매일의 기록이 쌓이면 "나만의 감정-음식 패턴" 분석 리포트를 제공한다.

### 핵심 가치

- "오늘 뭐 먹지?" 해결 — 기분/상황 기반의 새로운 메뉴 결정 방식
- 감성 레시피 카드 — SNS 공유를 통한 자연스러운 바이럴
- 데이터 축적 — 장기적으로 감정-음식 패턴 분석 리포트 제공

### 타겟 사용자

- 1차: 자취생, 요리 초보 (20~30대)
- 2차: 건강/다이어트 관심층
- 3차: 요리 콘텐츠 소비자 (SNS 활발 사용층)

### 경쟁 현황

| 서비스 | 기분 기록 | AI 레시피 추천 | 기분→레시피 연결 | 패턴 분석 | 한국 음식 특화 |

## 2. 사용자 플로우

### 핵심 시나리오

```
[기분 입력] → [상황 컨텍스트] → [AI 레시피 생성] → [감성 레시피 카드] → [공유 / 저장]
                                                                              ↓
                                                              [다이어리 축적 → 패턴 분석]
```

### 상세 플로우

1. **기분 입력**: 이모지 선택 또는 자유 텍스트 입력 (예: "피곤하고 위로받고 싶어")
2. **상황 컨텍스트 (선택)**: 날씨, 누구와 먹는지, 시간대, 보유 재료
3. **AI 레시피 생성**: OpenAI API가 기분+상황을 분석하여 맞춤 레시피 + 감성 메시지 생성 (Moodish 톤 앤 매너 적용)
4. **결과 카드 출력**: 레시피명, 재료, 조리법, 감성 한마디가 담긴 시각적 카드
5. **공유/저장**: SNS 공유 (인스타, 카카오톡) 또는 다이어리에 저장
6. **패턴 분석**: 데이터 축적 후 주간/월간 감정-음식 리포트 제공

## 4. AI 엔진 설계

### 사용 모델

- **OpenAI API** (GPT-4o 또는 GPT-4o-mini)
- 레시피 생성: 기분 + 상황 + 재료를 입력으로 받아 구조화된 JSON 응답 반환
- 감성 메시지: 사용자 기분에 공감하는 따뜻한 한마디 생성

### 프롬프트 구조 (예시)

```
역할: 당신은 한국 가정식 전문 감성 셰프입니다.
입력: 사용자 기분, 상황, 보유 재료
출력 형식 (JSON):
- reaction: 사용자 기분에 공감하는 따뜻한 한마디
- recipe_name: 감성적인 레시피 이름 (예: "퇴근길 위로 된장찌개")
- ingredients: 재료 목록 (분량 포함)
- steps: 조리 단계
- cooking_time: 예상 조리 시간
- difficulty: 난이도 (쉬움/보통/어려움)
- description: 이 요리가 오늘 기분에 어울리는 이유
```

### API 호출 전략

- 감성 메시지(reaction)와 레시피(recipe)를 병렬 호출하여 UX 대기 시간 최소화
- 스트리밍 응답으로 감성 메시지를 먼저 표시, 레시피 로딩 중 대기감 감소
- GPT-4o-mini를 기본으로 사용하여 비용 최적화, 프리미엄 사용자에게 GPT-4o 제공

| 영역 | 기술 | 비고 |

## 7. 수익 모델

### 1단계: SEO 광고 수익

- AI가 생성한 레시피를 SEO 최적화된 정적 페이지로 자동 변환
- 롱테일 키워드 타겟: "스트레스 받을 때 먹으면 좋은 음식", "비 오는 날 레시피" 등
- 구글 애드센스 연동

### 2단계: 프리미엄 구독

- 무료: 일 3회 레시피 생성, 기본 다이어리
- 프리미엄 (월 3,900원~): 무제한 생성, 고급 패턴 분석, 맞춤 식단 추천, 광고 제거

### 3단계: 제휴 커머스

- 레시피 재료 → 쿠팡파트너스, 마켓컬리 등 연동
- 재료 구매 링크를 통한 제휴 수수료
